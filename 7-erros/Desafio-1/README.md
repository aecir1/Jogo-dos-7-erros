# Desafio 1 - Perfil Profissional

Neste desafio, você recebeu um arquivo HTML parcialmente quebrado (`index.html`). O objetivo é utilizar os conhecimentos de HTML, CSS e as ferramentas de desenvolvedor do navegador (F12) para encontrar e consertar os **7 erros propositais** escondidos no código.

O resultado esperado após todas as correções deve ser visualmente idêntico ao arquivo `gabarito.html`.

## Como praticar
1. Abra o arquivo `index.html` em seu navegador.
2. Pressione F12 para abrir o DevTools.
3. Utilize as abas *Elements* e *Console* para identificar os problemas de layout e comportamento.
4. Altere o código fonte utilizando o seu editor de código favorito (ex: VS Code).

*Consulte as soluções gerais na raiz do projeto se precisar de ajuda!*
