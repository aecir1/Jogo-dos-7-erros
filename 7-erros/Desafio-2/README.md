# Desafio 2 - Portal de Autenticação

Neste desafio, você recebeu uma página de login (`index.html`) que contém problemas estruturais e de lógica. O objetivo é utilizar o DevTools (F12) para encontrar e consertar os **7 erros propositais** que quebram o layout e o envio do formulário.

O resultado esperado após todas as correções deve ser visualmente idêntico e ter o mesmo comportamento do arquivo `gabarito.html`.

## Como praticar
1. Abra o arquivo `index.html` em seu navegador.
2. Pressione F12 para abrir o DevTools.
3. Utilize a aba *Elements* para corrigir os problemas visuais (caixas e textos apertados).
4. Utilize a aba *Console* e o depurador para consertar o JavaScript que não está conseguindo impedir o recarregamento da página no clique do botão.
5. Aplique as correções no arquivo fonte utilizando seu editor de texto.
